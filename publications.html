Conjecture M-1 — Convergence Function c(N)

The central finding of the program is that the minimum number of iterations for the statistical convergence of the Domus Horizon algorithm satisfies:

k*(N) = c(N) · N · log(N)

where c(N) is not a universal constant but a function that evolves with scale, governed by three distinct geometric phases.

The Three Complexity Phases

Phase I — Local Dominance  N ≤ 100

c = 6.000 = K₂ (Kissing Number of the Euclidean plane). Local Delaunay geometry dominates. Each node has 6 neighbors and SA needs exactly c = K₂ "rotations" per Voronoi neighborhood.

Transition — Cluster Saturation  100 < N < 1.000

c rises from 6 to ~23. High-density clusters emerge. SA must navigate between clusters, breaking the local planarity approximation. The energy landscape becomes rugged.

Phase II — Macro Self-Organization  N > 1.000

c decreases following c(N) = 160 / log(N). Uniform distribution generates statistical homogeneity. The KNN-12 graph is nearly regular and SA exploits this regularity: every 2-opt move carries predictable statistical information.

Logarithmic Runge Model (MRL)

As a unified functional form over all N we propose:

a · log(N) c(N) = ──────────────────── 1 + b · log(N)² Fitted parameters: a = 2.946 · b = 0.02229 Analytical maximum: N* = exp(1/√b) ≈ 811 → c(N*) ≈ 9.87

The model qualitatively captures the three phases but the Two-Phase Model M2F (c₀ = 6 for N ≤ 100 · linear-log transition · 160/log(N) for N > 1.000) offers a better empirical fit.
