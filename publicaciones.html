Publicación 1 — Framework Domus Horizon Cognitivo

Resultados principales del Programa M-1: la función de convergencia c(N), las tres fases geométricas, el modelo de Runge, el Oráculo Geométrico Determinista y el experimento mega-escala Quark-Cuaternario.

Conjetura M-1 — Función de Convergencia c(N)

El hallazgo central del programa es que el número mínimo de iteraciones para la convergencia estadística del algoritmo Domus Horizon satisface:

k*(N) = c(N) · N · log(N)

donde c(N) no es una constante universal sino una función que evoluciona con la escala, gobernada por tres fases geométricas distintas.

Las Tres Fases de Complejidad

Fase I — Dominancia Local  N ≤ 100

c = 6.000 = K₂ (Kissing Number del plano euclidiano). La geometría local Delaunay domina. Cada nodo tiene 6 vecinos y el SA necesita exactamente c = K₂ "rotaciones" por vecindad de Voronoi.

Transición — Saturación de Clústeres  100 < N < 1.000

c sube de 6 a ~23. Emergen clústeres de alta densidad. El SA debe navegar entre clústeres, rompiendo la aproximación de planaridad local. El paisaje de energía se vuelve rugoso.

Fase II — Autoorganización Macro  N > 1.000

c decrece siguiendo c(N) = 160 / log(N). La distribución uniforme genera homogeneidad estadística. El grafo KNN-12 es casi-regular y el SA explota esta regularidad: cada movimiento 2-opt porta información estadística predecible.

Modelo de Runge Logarítmico (MRL)

Como forma funcional unificada sobre todos los N se propone:

a · log(N) c(N) = ──────────────────── 1 + b · log(N)² Parámetros ajustados: a = 2.946 · b = 0.02229 Máximo analítico: N* = exp(1/√b) ≈ 811 → c(N*) ≈ 9.87

El modelo captura cualitativamente las tres fases pero el Modelo de Dos Fases M2F (c₀ = 6 para N ≤ 100 · transición lineal-log · 160/log(N) para N > 1.000) ofrece mejor ajuste empírico.

Oráculo Geométrico Determinista (DGO)

El DGO es un algoritmo completamente determinista basado en triangulación de Delaunay + búsqueda 2-opt restringida. Elimina el 94.3% del espacio de búsqueda para N = 100 y supera al SA estocástico en 4 de 5 instancias.
